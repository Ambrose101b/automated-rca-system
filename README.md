# Automated Root Cause Analysis (RCA) System

## Overview
This project simulates a production environment where an automated system monitors a service, detects failures, gathers system metrics (CPU, Memory, Logs), and generates a Root Cause Analysis report to accelerate incident response.

## Tech Stack
* **OS:** Linux (Ubuntu)
* **Scripting:** Python
* **Containerization:** Docker
* **CI/CD:** Jenkins, GitHub Actions
